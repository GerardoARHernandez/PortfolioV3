##  Características

- ✅ **Diseño Responsive** - Adaptado a todos los dispositivos
- ✅ **Modo Oscuro** - Diseño moderno con colores oscuros
- ✅ **Animaciones Suaves** - Transiciones y efectos visuales atractivos
- ✅ **SPA (Single Page Application)** - Navegación fluida sin recargas
- ✅ **Componentes Reutilizables** - Código limpio y mantenible
- ✅ **Formulario de Contacto** - Integración con mailto:
- ✅ **Galería de Proyectos** - Filtros y búsqueda de proyectos
- ✅ **Descarga de CV** - Botón para descargar currículum
- ✅ **Optimizado para SEO** - Meta tags y estructura semántica

## 🛠️ Tecnologías Utilizadas

### Frontend
- **React 18** - Biblioteca para construir interfaces de usuario
- **React Router DOM 6** - Enrutamiento para aplicaciones SPA
- **Tailwind CSS 4** - Framework de CSS utilitario
- **Vite** - Build tool rápido y moderno

### Íconos y Estilos
- **React Icons** - Biblioteca de íconos populares
- **Heroicons** - Íconos SVG hermosos por Tailwind Labs
- **CSS Modules** - Estilos scoped por componente
